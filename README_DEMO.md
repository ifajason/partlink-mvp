# PartLink MVP - 可演示原型

## 項目概述

PartLink 是一個為修配廠和零件商設計的快速上架、集中管理詢價的SaaS平台。

## 核心功能

### ✅ 已實現的功能

#### 賣家端
1. **認證系統**
   - 手機號碼 + OTP驗證登入
   - 測試OTP：`123456`
   - 商家信息設置（名稱、郵箱、地址、營業時間）

2. **儀表板**
   - 數據統計（總上架、總詢價、待回覆、已成交）
   - 最近詢價列表
   - 快速操作按鈕

3. **快速上架**
   - 語音輸入（使用Web Speech API，支持繁體中文）
   - 照片上傳（最多3張）
   - 自動解析（品牌、年份、零件名稱、狀況）
   - 手動修正
   - 生成詢價連結

4. **詢價管理**
   - 按狀態分類（待回覆/已回覆/已成交）
   - 複製買家聯絡方式
   - 標記已回覆/已成交
   - 詢價詳情展示

#### 買家端
1. **詢價頁面**
   - 零件信息展示
   - 填寫姓名、聯絡方式、問題
   - 提交詢價

#### 系統功能
1. **品牌數據庫**
   - 50+個品牌
   - 100+個零件類型
   - 6種狀況分類

2. **數據存儲**
   - 使用LocalStorage（無需後端）
   - 自動保存和恢復

## 使用指南

### 1. 登入
- 手機號碼：任意（例如：0912345678）
- OTP驗證碼：`123456`
- 填寫商家信息後登入

### 2. 快速上架
- 點擊「快速上架」按鈕
- 選擇語音輸入或手動輸入
- 說出或填寫：品牌、年份、零件名稱、狀況
- 上傳照片（可選）
- 完成上架，獲得詢價連結

### 3. 分享詢價連結
- 複製生成的詢價連結
- 分享到Facebook、LINE、Instagram等任何平台
- 買家點擊連結填寫詢價

### 4. 管理詢價
- 查看所有詢價
- 複製買家聯絡方式
- 標記已回覆或已成交

## 技術棧

- **前端框架**：React 19 + TypeScript
- **UI組件**：shadcn/ui + Tailwind CSS 4
- **路由**：Wouter
- **狀態管理**：React Context
- **語音識別**：Web Speech API
- **存儲**：LocalStorage

## 演示流程

### 場景1：新商家上架
1. 登入系統
2. 點擊「快速上架」
3. 使用語音輸入：「Altis 2017 左前大燈 原廠 有裂痕」
4. 系統自動解析品牌、年份、零件、狀況
5. 上傳照片
6. 完成上架，獲得詢價連結

### 場景2：買家詢價
1. 買家收到詢價連結
2. 點擊連結進入詢價頁面
3. 填寫姓名、聯絡方式、問題
4. 提交詢價

### 場景3：商家回覆
1. 商家登入系統
2. 在儀表板看到新詢價
3. 進入「詢價管理」頁面
4. 複製買家聯絡方式
5. 通過LINE或電話回覆買家
6. 標記「已回覆」或「已成交」

## 部署

### 本地運行
```bash
cd /home/ubuntu/partlink-demo
pnpm install
pnpm dev
```

### 生產部署
```bash
pnpm build
pnpm start
```

## 注意事項

1. **語音識別**：需要使用支持Web Speech API的瀏覽器（Chrome、Edge、Safari）
2. **LocalStorage**：數據存儲在瀏覽器本地，清除瀏覽器數據會丟失
3. **OTP驗證**：當前為演示模式，實際部署需要集成真實OTP服務
4. **詢價連結**：當前使用時間戳生成，實際部署需要改進
